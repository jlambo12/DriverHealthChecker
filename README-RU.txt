Driver Health Checker MVP

Что внутри:
- WPF приложение для Windows
- сканирование важных драйверов
- фильтрация: GPU / сеть / хранение / аудио
- статус по дате
- повторное сканирование
- пометка "Недавно обновлён", если версия или дата драйвера изменилась
- кнопка открытия официального источника для части сценариев

Как запустить:
1. Открой PowerShell в корне проекта
2. Выполни:
   dotnet restore
   dotnet run --project .\DriverHealthChecker.App\

Если у тебя уже есть свой проект:
- можешь просто заменить файлы в папке DriverHealthChecker.App
- либо заменить всю папку проекта целиком
