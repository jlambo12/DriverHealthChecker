# DriverHealthChecker

## Быстрый старт окружения

Для автоматической подготовки окружения на Windows используйте:

```powershell
.\scripts\setup-dev-env.ps1
```

Скрипт установит .NET SDK 10 (если не найден), проверит конфигурацию проекта и выполнит восстановление зависимостей.

Подробный аудит и roadmap: `docs/PROJECT_AUDIT_AND_ROADMAP.md`.

Безопасный процесс по веткам/PR/merge: `docs/SAFE_PR_WORKFLOW_RU.md`.
